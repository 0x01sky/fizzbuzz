# fizzbuzz 🔊
A popular coding problem coded in 5 different programming languages !

# How it works?
  - so this program checks if any number less or equal than the number that the user typed in the input, if the number is a multiple of 3 , the output would be Fizz, if it's a multiple of 5 , the output would be Buzz, if the number is a multiple of both, the output is FizzBuzz , if the number is not a Fizz nor a Buzz nor a Fizzbuzz , it output is a number!
    
  - Time Complexity : O(n)
